상속관계가 성립해야 다형성을 이용 가능
모든 클래스는 오브젝트 클래스에게 상속 받았음
프로텍티드나 초기화블록 등은 상속이 불가하고 그 클래스 내에서만 사용한다.
다형성: 부모의 참조변수로 여러 자식객체를 가리키는 것
instanceof: 참조변수에서 형변환이 가능한지 확인하는 키워드
예외처리 시 세부사항을 위에서 캐치하고 아래로 내려갈수록 더 큰 범위를 캐치

클래스에는 크게 4가지가 들어감( 필드, 생성자, 메소드, 초기화블럭 )

인스턴스 변수는 객체가 생성되고 종료 될 때까지만 존재
스태틱 변수는 항상 존재

//인스턴스 메소드 : 메모리에 없기 때문에 인스턴스, 스태틱 변수 모두 사용 가능
//스태틱 메소드 : 메모리에 올라와 있기 때문에 인스턴스 변수는 객체 생성 후 사용가능

중첩 클래스 내부 클래스에는 static을 붙일 수 있음

class 앞에 final을 붙이면 상속(extends) 금지
final이 붙은 메소드는 재정의(Override) 금지
변수에 static final이 붙으면 상수 (값의 변경 금지)
